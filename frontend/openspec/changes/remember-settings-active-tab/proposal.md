# 记住设置页当前 Tab

## 背景

用户从设置页的“AI助手”Tab 返回首页后，再次进入设置页时，页面会回到默认“环境”Tab。用户期望设置页能记住上次停留位置。

## 目标

- 设置页切换 Tab 后记住当前 Tab。
- 从其他页面重新进入设置页时，恢复上次停留的 Tab，例如“AI助手”。
- 仅保存非敏感 UI 偏好，不保存模型配置、API Key 或业务数据。

## 非目标

- 不修改路由结构。
- 不新增后端接口。
- 不改变设置页各 Tab 内部表单状态持久化规则。

## 影响范围

- `frontend/src/pages/SettingsPage.tsx`

## 风险等级

低风险。该变更只持久化设置页当前 Tab 的 UI 偏好，不触发本机高风险动作。

## 验收标准

- 进入设置页并切换到“AI助手”。
- 切换到首页后再次进入设置页，仍停留在“AI助手”。
- 存储值异常时回退到“环境”。
- `pnpm run typecheck` 通过。
