# 同步默认 AI 模型到首页对话

## 背景

设置页点击“设为默认”后，设置页模型卡片会更新默认标记，但首页 AI 对话工具栏仍可能保留旧的当前模型，导致“使用 AI 模型”显示与最新默认模型不一致。

## 目标

- 设置页模型“设为默认”成功后，同步刷新全局 AI 助手模型列表。
- 首页和右侧 AI 助手的当前模型切换到新的默认模型。
- 将新的默认模型同步到后端运行时模型。

## 非目标

- 不修改后端默认模型接口契约。
- 不新增运行时模型查询接口。
- 不调整模型创建、编辑、删除流程。

## 影响范围

- `frontend/src/layouts/AssistantPanelContext.tsx`
- `frontend/src/pages/SettingsPage.tsx`

## 风险等级

低风险。该变更只同步前端模型选择状态和已有运行时模型切换接口，不执行本机高风险动作。

## 验收标准

- 设置页点击“设为默认”成功后，全局当前模型切换为该模型。
- 返回首页后，AI 对话工具栏显示新的默认模型名称。
- 后端运行时模型切换接口使用该模型 ID。
- `pnpm run typecheck` 通过。
