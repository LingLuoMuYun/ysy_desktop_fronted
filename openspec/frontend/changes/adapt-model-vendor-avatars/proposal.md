# 模型厂商头像适配

## 背景

设置页模型管理中，模型卡片和详情摘要当前使用统一的 Bot 渐变图标，无法体现模型厂商。用户提供了厂商图标资源，希望按模型厂商展示对应头像。

## 目标

- 将提供的厂商图标复制到前端静态资源目录。
- 模型管理列表卡片按模型厂商展示对应图片头像。
- 模型详情、编辑摘要按同一规则展示厂商图片头像。
- 未匹配到厂商图标时保留现有 Bot 渐变兜底。

## 非目标

- 不修改模型厂商后端枚举。
- 不修改模型列表、保存、检测接口。
- 不处理未提供图标的 `custom` / OpenAI 兼容厂商。

## 风险等级

低风险。仅调整前端静态资源和展示逻辑。

## 验收标准

- DeepSeek、MiniMax、小米、智谱、月之暗面、火山引擎、硅基流动、阶跃星辰、阿里云对应模型显示厂商图片头像。
- 阿里云百炼等阿里云相关展示映射为阿里云图标。
- 未匹配厂商显示原有 Bot 兜底头像。
- TypeScript 类型检查通过。
