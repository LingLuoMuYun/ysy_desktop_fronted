# Acceptance: frontend-page-framework

## 验收范围

本验收只覆盖前端页面框架、静态结构、mock 数据和视觉还原。不验收真实后端、真实 Electron 本机能力、真实 AI 对话或真实任务执行。

## 验收清单

- [x] Change 文件位于 `openspec/frontend/changes/frontend-page-framework/`。
- [x] 前端应用可以启动并打开首页。
- [x] 左侧导航展示首页、项目、任务、数据、模型、设置，并支持切换。统一顶部栏提供侧边栏收起/展开按钮。
- [x] 统一顶部栏：首页展示历史对话、新对话、打开位置和资源概览入口；其他模块展示当前模块名和 AI 助手开关。
- [x] 首页展示中心标题、AI 输入框、辅助模式、模型选择、项目关联、发送按钮和建议问题；资源概览在右侧上下文栏展示。
- [x] 项目页面展示创建项目入口、筛选区、项目列表/卡片、状态标签和行级操作，内容区不重复展示“项目”标题。
- [x] 任务页面展示训练/部署切换、创建入口、筛选区、任务列表、进度、状态和行级操作，内容区不重复展示“任务”标题。
- [x] 数据页面展示导入入口、筛选区、数据集列表、路径、格式、样本、关联项目和行级操作，内容区不重复展示“数据”标题。
- [x] 模型页面展示导入入口、筛选区、模型列表、路径、类型、来源、关联项目和行级操作，内容区不重复展示“模型”标题。
- [x] 设置页面展示环境、AI 助手、个人信息分区，以及环境列表，内容区不重复展示“设置”标题。
- [x] 非首页页面展示右侧 AI 助手面板或明确占位，设置页不再内部嵌套第二个 AI 面板。
- [x] 高风险操作入口不会直接执行真实动作。
- [x] API Key、token、环境变量和敏感本地路径不会明文展示。
- [x] 页面没有明显文本溢出、遮挡、错位或布局跳动。
- [x] 静态检查通过。
- [x] 构建检查通过。
- [x] 浏览器视觉验证完成，并记录视口和结果。

## 原型对齐点

- 首页对齐 `http://10.0.89.199:5173/` 的居中输入体验。
- 项目页对齐 `/projects` 的项目卡片和筛选结构。
- 任务页对齐 `/tasks` 的训练/部署任务列表结构。
- 数据页对齐 `/data` 的数据集列表结构。
- 模型页对齐 `/models` 的模型资产列表结构。
- 设置页对齐 `/settings` 的设置分区和环境列表结构。

## 验收记录

- 静态检查：`pnpm run typecheck` 通过。
- 构建检查：`pnpm run build` 通过。
- 本地服务：`pnpm run dev`，地址 `http://localhost:5174/`。
- 浏览器验证视口：约 1280x720。
- 路由验证：首页 `/`、项目 `/projects`、任务 `/tasks`、数据 `/data`、模型 `/models`、设置 `/settings` 均可访问。
- 导航验证：点击“项目”后 URL 切换为 `/projects`，导航选中态和统一顶部栏模块名同步。
- 标题去重验证：项目页内容区 `h1` 数量为 `0`，设置页内部 `settings-topbar h1` 数量为 `0`，模块名保留在统一顶部栏。
- 控制台错误：未发现 error 级别日志。
