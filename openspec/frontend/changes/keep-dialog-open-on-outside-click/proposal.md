# keep-dialog-open-on-outside-click

## 背景

设置页多个弹窗在遮罩层绑定了点击关闭逻辑。用户点击弹窗外区域时会直接退出弹窗，容易造成表单编辑、确认流程或删除范围选择被中断。

## 目标

- 鼠标点击弹窗外遮罩区域时，不关闭当前弹窗。
- 保留明确退出入口，包括右上角关闭、取消、返回等按钮。
- 不改变弹窗内提交、确认、返回、删除等业务动作。

## 非目标

- 不调整弹窗视觉样式。
- 不改变后端 API、数据结构或风险确认分级。
- 不新增全局弹窗框架。

## 影响范围

- 前端设置页：环境创建/确认/成功弹窗、模型新增/详情/编辑弹窗、环境删除确认弹窗、通用删除确认弹窗。

## 验收标准

- 点击弹窗外遮罩区域时，弹窗保持打开。
- 右上角关闭按钮、取消按钮、返回按钮仍可退出或返回上一步。
- TypeScript 类型检查通过。
